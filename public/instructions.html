How to enter a winning ticket number

  1. Tap any basket cell on the grid.
  2. Type the winning ticket number from the drawn ticket.
  3. (Optional) Add a description like the basket name.
  4. Tap Save.

The display board on the TV updates within a second. Customers scanning the QR code see the number on their phone instantly.

When a winner picks up their basket

  1. Tap the same basket cell again.
  2. Check the "Picked up by winner" toggle.
  3. Tap Save.

The cell turns green on the display so customers know it's claimed.

👥 Multiple helpers can enter at the same time

Anyone with the URL and PIN can enter ticket numbers on their own phone simultaneously. Everyone sees each other's entries appear live. No setup needed — just share the URL and PIN with each helper.

Basket types you'll see

  • Gold — Regular Basket
  • Orange — Big Ticket Item
  • Purple — Special
  • Green — Already picked up
